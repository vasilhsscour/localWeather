# localWeather
Local Weather

This page take the position of the user in (longitude and latitude)
Connect with an api with this coordinates and take the results.
The results is :
  the temperature in celsius,
  the max and min temperature,
  the speed and the degrees of the wind,
  the humidity,
  the weather and the number of clouds,
  an image for the weather,
  and the location.
  
Also i make some objest which depends on the degrees of the wind returns the direction,
                              depends on the speed returns the discription,
                              and depends on the weather i return an image relative to it.

The results go in a box.
This box have also two buttons one active and one inactive.
this buttons is to change the temperature for Celsius to fahrenheits and opposite.
